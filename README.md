# Wantedly

## クラウドインフラ、バックエンド開発を得意とするエンジニア

```
株式会社Freedoming所属の岸伸考です。
クラウドインフラの設計・構築、バックエンド開発を専門にしています。

AWS と Java/Spring Boot を中心に、インフラ構築からアプリケーション開発まで幅広く対応できることが強みです。
バックエンドを主軸としていますが、API 連携を踏まえたフロントエンドの実装経験もあります。

小規模〜中規模のチームで、複数の業務を柔軟に兼任することで力を発揮できます。

■得意分野
・クラウドアーキテクチャ設計
・クラウドインフラのTerraform化（AWS、Azure）
・サーバレス環境での開発
・アプリのDocker化
・スクリプトによる自動化（Git、DB操作、ログ収集など）

■主な技術スタック
# クラウド
　AWS（ECS/Lambda/ALB/EC2/EventBridge/StepFunctions/SQS/DynamoDB/Aurora/S3/Athena/CodePipeline）
　Azure（ACA/VM/Bastion/AppGW/Vnet/NSG/PrivateDNS/PostgreSQL FlexServer/ACR）
# 言語
　Java/JavaScript/Python/ShellScript/HCL
# フレームワーク
　Spring Boot/Spring/Struts/Vue.js
```
